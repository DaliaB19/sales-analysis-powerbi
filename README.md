# 📊 Power BI Sales Analysis — Projet Data Analyst

## 🎯 Objectif du projet

Ce projet a pour objectif d’analyser des données de ventes afin de mettre en évidence les performances commerciales, les tendances et les indicateurs clés de performance (KPI).

---

## 📌 Contexte

Dans un contexte business, l’analyse des ventes permet de :

* comprendre les performances produits
* identifier les tendances de consommation
* améliorer la prise de décision stratégique

---

## 📊 Contenu du projet

Ce repository contient un rapport Power BI complet :

👉 📄 Fichier Power BI :
[`PowerBI_Sales Rapport.pbix`](https://github.com/DaliaB19/sales-analysis-powerbi/blob/main/PowerBI_Rapport_Sales.pbix)

## 📈 Analyses réalisées

* Analyse du chiffre d’affaires
* Analyse des ventes par produit
* Analyse des tendances temporelles
* Identification des produits les plus performants
* Suivi des KPI commerciaux
